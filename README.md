# EPC Industry Analytics - Summer Internship 2026

Assignment notebook for the IDEAS Summer Internship Program 2026.

## What's here

- **`13_EPC_Industry_Analytics_Assignment_Summer_2026.ipynb`** - The assignment notebook. Covers EDA on EPC industry data (cleaning, stats, outliers, feature engineering) and K-Means clustering on MNIST digits.
- **`EPC_Industry_Analytics_Demo.xlsx`** - The dataset used in the notebook.

## Topics covered

1. Data loading and exploration
2. Duplicates and missing value treatment
3. Correlation and outlier detection (IQR)
4. Group-based aggregation and feature engineering
5. K-Means clustering on MNIST with F1 score evaluation

## Requirements

- Python 3
- pandas, numpy, scikit-learn
